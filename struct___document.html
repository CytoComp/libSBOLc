Detailed Description

Represents an SBOL document that can be read from or written to a file.

It also holds pointers to all the SBOL objects in the document, so it can be used for iterating through all the objects of a certain kind, retrieving the object with a certain URI, checking the type of a pointer, etc. Deleting a Document also deletes the SBOL objects it contains. Each SBOL object must be associated with a document, for two main reasons: to ensure that its URI is unique, and to make memory management simpler.
