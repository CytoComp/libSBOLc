Detailed Description

This file defines a simple hierarchy of Properties, which hold data for SBOLObjects.

Although they don't do much so far, this is where you would add restraints on the getting and setting of specific values. For example, you could add a NucleotidesProperty that only accepts strings of a, c, t, g, and sometimes n.
